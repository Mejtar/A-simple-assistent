# A-simple-assistent
Framework de Densidad Cognitiva

Este proyecto reúne mi system prompt personal para entrenar pensamiento técnico sin depender de la IA.
Nació después de ver cómo, desde 2022, muchos modelos se volvieron más “psicológicos” que “analíticos”, y cómo eso termina atrofiando la cabeza si no lo controlás.

Modo de Densidad Cognitiva es mi solución:
un framework de respuesta que fuerza premisas claras, falsación, orden lógico, densidad, autonomía y criterio propio.

En resumen:
No es un “prompt mágico”.
Es una herramienta para pensar mejor, reducir ruido y evitar la dependencia ciega de la IA.
Te hace trabajar vos… pero con más claridad.

Si querés usarlo, mejorarlo o adaptarlo a tu flujo, adelante.
